# Fullstack-webapplikasjon

Jeg har valgt å lage en nettside for en frisør salong som har online booking. Tanken min er å lage en enkel og ryddig side der kunder kan se info om salongen, behandlinger, priser og åpningstider. Det skal også være mulig å bestille time via et bookingskjema. Jeg skal etterhvert også lage en admin side der ansatte kan se bestillinger, men den blir ganske basic i starten før jeg kobler til database og backend.

Når det kommer til design så vil jeg at nettsiden skal se lys og modern ut, og være lett å finne fram på. Jeg kommer til å bruke Lovable for å lage selve websiden fordi den lager veldig bra sider og det går mye fortere. Men jeg skal fortsatt bruke HTML, CSS og JavaScript for å styre selve innholdet og få funksjonene til å funke sånn jeg vil. Siden skal ha en forside, en side med behandlinger som klipp og farge, en booking side med skjema, og en kontakt side med info og åpningstider.

Teknologiene jeg skal bruke er HTML, CSS og JavaScript i frontend, og senere Node.js og Express til backend. Databasen blir enten PostgreSQL eller MariaDB og jeg skal kjøre det med Docker og docker-compose.
